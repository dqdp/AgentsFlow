# Domain Pack: agentic-system

## Purpose

Rules for systems with agents, tools, memory, context, model routing, policy, and traces.

## Rules

- Prompt changes require behavioral scenarios when they affect tool/memory/policy behavior.
- Memory writes must be policy-mediated.
- Tool calls should be traceable.
- Destructive operations require explicit approval.
- Model escalation should be observable in evidence or trace.


## Usage

Workflows may load this pack to parameterize contract writing, scenario design, impact mapping, and review.
